Git In Python - Git made easier!

Git In Python is a simple reimplementation of git for easier use and basic tasks to save time.

Here are some features of the program

The program supports git add and commits management

The program supports branching to orgainize workflow.

The program supports distrubted workflows with multiple developers.

About the project

I was reading about Vim and how it got started, what i found is that Vi [the original program] was a UNIX closed-source tool and Vim was a reimplementation of it. I was curious how do people rebuild such complex tools, so I wanted to experiment with something complex. Git is a simple tool when you use it but is very complex under the hood, so this is a fun challenge!
